### Запуск

Створити файл `.env` з необхідними ключами доступу:

```
MODAL_KEY=wk-xxx
MODAL_SECRET=ws-xxx
OPENAI_API_KEY=sk-proj-xxx
```

Встановити необхідні бібліотеки:

```
pip streamlit openai dotenv
```

Запустити:

```
streamlit run app.py
```

### Файлова структура

- `eval`: Тестові дані для порівняння моделей.
- `examples`: Приклади вхідних текстів.
- `generated`: Приклади згенерованих текстів.
- `grammars`: Граматики для структурованої генерації.
- `system_prompts`: Системні промпти.
- `app.py`: Користувацький інтерфейс.
- `deploy.py`: Хостинг `gpt-oss-120b`.
- `embedder.py`: Отримання ембедингів тексту.
- `eval.py`: Порівняння моделей для ненерації Прологу.
- `generate.py`: Генерація нових похідних текстів крізь Пролог.
- `llm.py`: Класи для взаємодії із великими мовними моделями.
- `resources.py`: Підвантаження текстових файлів.
- `utils.py`: Допоміжні функції.
