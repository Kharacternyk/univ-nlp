### Запуск

Створити файл `.env` з необхідними ключами доступу:

```
MODAL_KEY=wk-xxx
MODAL_SECRET=ws-xxx
OPENAI_API_KEY=sk-proj-xxx
```

Встановити необхідні бібліотеки:

```
pip streamlit openai dotenv
```

Запустити:

```
streamlit run app.py
```

Файл `app.py` містить коментарі.
